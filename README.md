# 不合格处理（nonconformance-handling）

面向质量工程师与 MRB 主持人的不合格品处置技能。规范返工/返修/让步/降级/报废判定与审批，生成不合格处置单与 MRB 评审记录，输出 **文字版(.txt) + Markdown(.md) 文档**（无网页/办公格式，重实效、不花哨）。

## 适用角色
- 质量工程师（QE）
- MRB 主持人（材料评审委员会）

## 核心能力
- 区分五种处置方式（返工/返修/让步/降级/报废）并给出适用条件、审批权限、风险提示。
- 生成结构化不合格处置单（隔离→处置→重检→最终处置）。
- 生成 MRB 评审记录（参会方、决议、责任、时限、验证）。
- 固定铁律提醒：返工≠返修、100% 重检、>2 次强制报废、让步三限制。

## 使用流程
1. 提供不合格信息：产品/缺陷/环节/数量/批次/行业。
2. 技能分类判定（致命/严重/轻微）、评估影响范围与紧急程度。
3. 给出处置建议（方式+条件+审批+风险）。
4. 生成处置单与 MRB 记录，调用 `scripts/build_report.py` 输出文字版与 MD 文档。

## 脚本说明（scripts/build_report.py）
- `python scripts/build_report.py` —— 使用内置小样本，产出 txt+md。
- `python scripts/build_report.py --data-file nc.json` —— 自定义数据。
- `python scripts/build_report.py --out-dir D:/临时` —— 指定输出目录（默认当前工作目录）。
- 输出：`nonconformance_report.txt`（纯文字版）与 `nonconformance_report.md`（Markdown）。无任何网页/办公格式。

## 能力边界
- 不替代工程/质量人员的技术判定（能否恢复、是否影响安全法规）。
- 不替代客户确认（让步涉及客户须书面确认）。
- 审批权限与判定标准为通用参考，以企业制度与客户特殊要求为准（标注 待企业补充）。

## 联动技能
- 表单生成（处置单复用记录模板）
- 计量管理（超期量具数据追溯）
- 8D / 根因分析（重复/批量不合格）
- 内审不符合项（体系性失效）
- 供应商管理（来料不合格预警）
